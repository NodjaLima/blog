Tragetória

Aplicando os conhecimentos já no primeiro dia, fizemos uma página HTML simples, sem uso de muitas propriedades CSS. Foi a partir dessa ideia que tive a coragem de iniciar esse projeto em que você agora me lê. Com essa pequena página aprendi os principios básicos de como estilizar paginas, com uso de background, alteração de cores, inserção de imagens e box model.
